Detailed Description

A DBusMenuExporter instance can serialize a menu over DBus.

◆ iconNameForAction()

virtual QString DBusMenuExporter::iconNameForAction ( QAction *  action)
protectedvirtual

Must extract the icon name for action.

This is the name which will be used to present the icon over DBus. Default implementation returns action->icon().name() when built on Qt >= 4.7 and a null string otherwise.

◆ setStatus()

void DBusMenuExporter::setStatus ( const QString &  status)

The status of the menu.

Can be one of "normal" or "notice". This can be used to notify the other side the menu should be made more visible. For example, appmenu uses it to tell Unity panel to show/hide the menubar when the Alt modifier is pressed/released.
